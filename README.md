# Recording Notes

A web application for taking timestamped notes during podcast, music, or video recording, with real-time collaboration. Export notes to REAPER, Audition, or Resolve to speed up post-production editing. Integrates with [SquadCast](https://squadcast.fm) and [Bitfocus Companion](https://companion.bitfocus.io).

<!-- screenshot: application overview / main view -->

## Quick Start

```bash
npx recordingnotes start
```

Or install globally:

```bash
npm install -g recordingnotes
recordingnotes start
```

Open `http://localhost:3000` in your browser.

## Features

- **Real-time collaboration** — multiple users annotate the same session simultaneously via WebSocket sync
- **Timestamped notes** — clock mode (time-of-day) or timer mode (relative to recording start)
- **Session management** — create, edit, and organize recording sessions
- **Export** — download as REAPER/Audition CSV markers or Resolve CMX3600 EDL
- **Integrations** — automated sessions via SquadCast webhooks and Bitfocus Companion triggers
- **Guest access** — shareable links for single-session collaboration
